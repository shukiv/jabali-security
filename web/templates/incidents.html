Security Incidents

Security events detected by the scanning engines (heuristic, entropy, YARA-X, ClamAV). Each scanner assigns a score to its findings, which are aggregated into a total threat score. Scoring thresholds determine the action: below 40 = ignored, 40–70 = logged as incident (low/medium), 70–100 = file quarantined (high), 100+ = account suspended (critical). Thresholds are configurable in Settings.
